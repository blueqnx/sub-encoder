**Subtitles UTF-8 Encoder**

Convert subtitles to standard utf-8 for using with media servers

![](https://cloud.githubusercontent.com/assets/2601749/15456976/8d058fbe-2080-11e6-8918-98f65a29a82b.png)

**Run Code**
```
npm install
bower install
nw ./
```

NW.js is required: 
http://nwjs.io/
